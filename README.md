# Politico
Politico is an application that enables users(voters) cast their votes for their respective candidate(politicians)

## Features
---
### Admin
- Create Party
- Modify Party
- Delete Party
- Create Offices

### Users
- Signup and Login
- Cast vote for respective candidate
- View Parties
- View offices

### Politician
- Aspire as candidate for an office
